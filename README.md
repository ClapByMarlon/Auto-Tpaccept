# Auto-Tpaccept

Ein LabyMod 3 Addon für Minecraft 1.8.9.

**Beschreibung:** Auto-Tpaccept nimmt Teleport-Anfragen von Spielern auf deiner Whitelist automatisch an.

## Funktionen

- Erkennt TPA- und TPAHere-Anfragen im Chat
- Nimmt nur Anfragen von Spielern auf der Whitelist an
- Delay ist einstellbar
- Spieler können in den LabyMod Addon-Einstellungen hinzugefügt und entfernt werden
- Pro Spieler kann TPA und TPAHere einzeln aktiviert oder deaktiviert werden

## Installation

1. Lade die neueste `.jar` aus den GitHub Releases herunter.
2. Lege die Datei in deinen LabyMod Addons-Ordner:

```text
.minecraft/LabyMod/addons-1.8/
```

Falls du MultiMC nutzt, kann der Pfad anders sein.

3. Starte Minecraft mit LabyMod 3 neu.
